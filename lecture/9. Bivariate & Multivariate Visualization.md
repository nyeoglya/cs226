### 9-1. Scatter Plot
- scatter plot: 각 $(x_i,y_i)$를 전부 plot한다. hue(색깔 지정)나 size(점 크기), style(찍는 점 모양)을 결정할 수 있다.
	- numerical-numerical에 사용할 수 있음.
	- slope랑 shape를 읽는다.
- line: 데이터가 정렬되어 있다. 가령, 시간이라던가.
	- slope나 curvature를 읽을 수 있다.
	- 정렬되지 않은 categories에 사용하면 안된다.
	- 데이터에 confidence interval을 만들 수 있음.
- regression: trend를 요약하기 위한 가장 간단한 선 긋기.
	- OLS, LOESS 같은 것들이 예시임.
- relplot: figure-level wrapper. subgroup 각각의 scatter plot임.
### 9-2. Multiple Distribution
- displot: histogram by groups
- kdeplot: bivariable data에 쓰임. stacked area plot. 각 dist.의 모양을 보여주지 않는다.
- pair plot(scatter-matrix): 가장 중요. 굉장히 유용함.
	- 각 grid가 multivariate feature를 보여준다.
	- 대각 성분은 univariate feature임.
	- 대각이 아닌 부분은 scatter plot이고, 대각 부분은 histogram.
- clustermap: 유사한 row/col이 비슷한 위치에 오게 하는 heatmap.
	- correlation을 계산한다.
	- 대칭적이고 대각성분이 전부 1.
- hexbin/2d hist: 데이터를 grid에 대해 그린다.
	- 굉장히 dense한 scatter cloud에 유용하다.
	- seaborn의 joint plot을 쓴다.
- 2d KDE: 2차원으로 정의를 늘린거임.
	- 2d gaussian curve를 더한 것이 하나의 예시임.
	- peak를 읽으면 됨.
	- seaborn의 joint plot을 쓴다.
- bubble: scatter plot의 특별한 형태. 데이터의 크기가 marker size에 그대로 드러남.
- box/violin/boxen: 각 class에 대해 그냥 box/violin/boxen를 따로 그리는거임.
- strip plot: 각 class에 대해 점을 따로 찍는다. 약간의 오차를 넣어서 구분이 가능하게 한다.
- contingency heatmap: contingency table임.
	- category-category에 사용한다.
	- grid that counts the co-occurence of each pair of categories.
	- class랑 bin의 집합이 있다고 하면, 각 class마다 bin의 개수를 그린다.
	- 직사각형 형태의 맵이 됨.
- correlation heatmap: numerical relation pair의 linear correlation을 그린다.
	- pearson correlation 사용함.
		- -1은 perfect negative, 1이 perfect positive
	- multi-colinearity에 사용함.
- parallel coordinate: class당 각 feature의 값을 2차원으로 그리고, 그것들을 연결한다.
- residual: ground truth와 실제 값의 차이(error)만을 계산해서 그리는거임. 별로 중요한 건 아님.
